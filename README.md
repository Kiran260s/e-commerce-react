# E-Commerce React Application

A modern e-commerce frontend built using React, React Router, Context API, Axios, and Tailwind CSS.

## Features

* Product Listing
* Product Search
* Category Filtering
* Shopping Cart Management
* Quantity Controls
* LocalStorage Persistence
* Responsive Design

## Tech Stack

* React
* React Router
* Axios
* Context API
* Tailwind CSS
* Vite

## Installation

```bash
git clone https://github.com/Kiran260s/e-commerce-react.git
cd e-commerce-react
npm install
npm run dev
```
